# 🍎🍌🍇 Image Classifier for fruits

## 📌 Overview
This project implements a **Convolutional Neural Network (CNN)** to classify fruit images into different categories.  
It demonstrates the full pipeline: **dataset preparation, model training, evaluation, and visualization of results**.

---

## ⚙️ Features
- Preprocessing of fruit images (resizing, normalization, augmentation)  
- CNN architecture built with **TensorFlow/Keras**  
- Training and validation with accuracy and loss graphs  
- Evaluation using **confusion matrix** and **classification report**  
- Sample predictions with visualization
